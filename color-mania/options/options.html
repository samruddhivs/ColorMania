    • copyToClipboard(text, element)
      Copies text to the clipboard and updates the UI to show "Copied!" temporarily.
    • exportColor()
      Exports the list of picked colors as a text file (`Colors.txt`) for download.
    • createColorPopup(color)
      Creates a popup displaying detailed information about a selected color, including Hex and RGB values.
    • showColors()
      Updates the UI to display all currently picked colors with interactive elements for each color.
    • hexToRgb(hex)
      Converts a hexadecimal color code (`hex`) into its RGB equivalent.
    • activateEyeDropper()
      Activates the eye dropper tool for users to select a color from the screen, adds it to the list of picked colors, and updates the UI.
    • clearAll()
      Clears all picked colors from the list and updates the UI accordingly.
